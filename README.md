## 🛑 Le Problème
L'abonnement gratuit Systeme.io vous limite à 3 tunnels de vente. Que faire si vous voulez promouvoir 10, 20 ou 50 produits différents en affiliation (1TPE, Systeme.io, etc.) ? De plus, les redirections simples sont pénalisées par Google et ignorées par les IA de recherche.

## ✅ La Solution : Le "Static Funnel Engine"
Ce dépôt vous permet de déployer une page de vente ("Bridge Page") autonome, hébergée gratuitement sur GitHub Pages.

* ⚡ **Performance 100/100 :** Fichier statique ultra-léger sans base de données.
* 🧠 **IA-Ready :** Injection native de microdonnées (JSON-LD) pour être référencé par les nouveaux moteurs de recherche (SearchGPT, Bing).
* 🛠️ **No-Code :** Vous n'avez qu'un seul fichier texte à modifier, le script s'occupe du reste du design.

## ⚙️ Comment déployer votre page en 2 minutes ?

### Étape 1 : Copiez ce moteur
Cliquez sur le bouton **"Fork"** en haut à droite de cette page pour copier ce projet sur votre propre compte GitHub.

### Étape 2 : Connectez votre compte Systeme.io
*Vous n'avez pas encore de compte Systeme.io ? [Créez votre compte gratuit ici pour récupérer vos commissions](https://systeme.io/?sa=sa16047325683a6745455482376be9c81d).*

### Étape 3 : Personnalisez votre offre
Dans votre nouveau dépôt, ouvrez le fichier `config.js` et cliquez sur l'icône du crayon pour le modifier. 
Remplissez les informations entre les guillemets (Vos liens d'affiliation, votre texte de vente, le nom du produit). Validez avec "Commit changes".

### Étape 4 : Mettez votre page en ligne
1. Allez dans l'onglet **"Settings"** (Paramètres) de votre dépôt.
2. Dans le menu de gauche, cliquez sur **"Pages"**.
3. Sous "Source", sélectionnez la branche **"main"** et cliquez sur Save.
4. Patientez 1 minute. GitHub vous donnera le lien public gratuit de votre nouvelle page de vente !
